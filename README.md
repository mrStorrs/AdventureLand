## Adventure Land - The Code MMORPG - Scripts

### Description:
Adventure Land is a game where you can completely automate every aspect of the game play.
Every single thing that can be done manually can be automated. You can choose to play with
manual controls, completely automated (through handwritten scripts) or a combination of both.

My goal is to completely Automate every part of the game. As i continue to create and refine
my scripts I will upload them here. 

### Running all characters on a single config
I find it cumbersome to try and switch through each character, just to change what they
were doing. Since that is far from Automated, I have decided to create a single script that
all characters can run. In this script, objects for each character will be created and 
you can choose what they will do here. The game will automatically reload a script if 
it sees that it was changed on another instance. So, all you will have to do to change the
behavior of a character will be to pop open the config script on any char, modify some 
variables/parameters. Then bam, the rest is script magic

### If you are planning to use this script for your own game:
The main config is where the magic happens. I have tried to include detailed enough
descriptions so that you can modify whatever parameters you may want to get 
your characters up and running! To do this you will need to first go into your
CODE editor in game. Then make a script for each of the ones you plan to run
(minimum is config, and combat). Once that is done, make sure that your config has
the script names inside of the "activities" area. Then just run the config script!

### Current status and functionality
Currently the config oriented script has the functionality to fully automate 
mob farming by simply entering the mob to farm into the config. I am working on
transitioning some merchanting scripts into the config oriented approach.
